This is a Node.js RESTful web service. No data is stored.
Account

GET /api/account Returns account data

POST /api/account/register Sends email to confirm registration
Body { email: string, name: string password: string, confirmPassword: string }

POST /api/account/login Returns jwt
Body { email: string, password: string }

Orders

GET /api/orders Returns all orders

GET /api/orders/:orderId Returns order by Id

POST /api/orders (FormData) Post data with file
Body { productId: int, quantity: int, hasFile: string ('true' || 'false') }

POST /api/orders/pdf Returns a pdf

POST /api/orders/zip Returns a zip file

DELETE /api/orders/:orderId Returns message with Id
